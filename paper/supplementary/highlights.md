# Highlights

• A complete Web GIS platform built with Flask, Leaflet.js, and PostgreSQL/PostGIS for real-time geospatial queries

• Demonstrates 93% performance improvement in proximity searches through proper GiST spatial indexing

• Provides an intuitive map-based interface for searching and visualizing establishments with geolocation support

• Open-source implementation with complete source code and documentation available for research community

• Practical case study using real-world data of pharmacies and gas stations in Brasília, Brazil
